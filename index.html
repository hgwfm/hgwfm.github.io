我是运维小铺，下边是需要用到的命令
复制代码后，粘贴到termux命令窗口中回车执行

一键安装alist的命令

pkg install alist && echo 'alist server &' >> ~/.bashrc && alist admin && alist server

————————————————————


如果一键安装alist不成功，则需要进行换源操作

一键换源命令

sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list

sed -i 's@^\(deb.*games stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable@' $PREFIX/etc/apt/sources.list.d/game.list

sed -i 's@^\(deb.*science stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science stable@' $PREFIX/etc/apt/sources.list.d/science.list

pkg update

上述命令，一次复制粘贴执行！
换源完成后，再次执行一键安装alist的命令

祝您成功！
————————————————————





苹果移动终端安装ISH模拟器后，需要安装APK的命令

# 下载
wget -qO- http://mirrors.aliyun.com/alpine/v3.12/main/x86/apk-tools-static-2.10.5-r1.apk | tar -xz sbin/apk.static
# 安装
./sbin/apk.static add apk-tools && rm sbin/apk.static && rmdir sbin 2> /dev/null



苹果系统APK安装完成后，再次执行一键安装alist的命令


pkg install alist && echo 'alist server &' >> ~/.bashrc && alist admin && alist server

祝您成功！
